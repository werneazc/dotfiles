# dotfiles

Private dot-files for shell and editor initialisations.

## neovim plugins

DejavuSansMono Nerd font is required.

### CoC extensions to install

Installation of nodejs12 or higher required. Installation of yarn required.

- coc-clangd
- coc-cmake
- coc-markdownlint
- coc-json
- coc-jedi
- coc-sh
- coc-yaml
